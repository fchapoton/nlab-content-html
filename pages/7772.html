Definition

(p. 39)

If kk is perfect any finite affine kk-group GG is in a unique way the product of four subgroups

G=a×b×c×dG=a\times b\times c\times d

where

  1. a∈Fem ka\in Fem_k is a formal étale multiplicative kk group.

  2. b∈Feu kb\in Feu_k is a formal étale unipotent kk group.

  3. c∈Fim kc\in Fim_k is a formal infinitesimal multiplicative kk group.

  4. d∈Fem kd\in Fem_k is a infinitesimal unipotent kk group.
