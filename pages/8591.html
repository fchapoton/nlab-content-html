Absolutely continuous measures

  • Definition
  • Generalisation
  • Defaults

Idea

A measure μ\mu is absolutely continuous with respect to ν\nu if we can think of μ\mu as a weighted variation on ν\nu.

Definition

Fix a measurable space XX and let μ\mu and ν\nu be two measures on XX.

Definition

The measure μ\mu is absolutely continuous with respect to ν\nu if every ν\nu-full set is also μ\mu-full.

Because null sets are more familiar than full sets, we may equivalently express things as follows (but this is not correct in constructive mathematics):

Definition (classical)

The measure μ\mu is absolutely continuous with respect to ν\nu if every ν\nu-null set is also μ\mu-null.

If μ\mu and ν\nu are positive measures, then we may also express this as follows:

Definition (positive, classical)

The positive measure μ\mu is absolutely continuous with respect to ν\nu if, for every measurable set AA, μ(A)=0\mu(A) = 0 if ν(A)=0\nu(A) = 0.

Since the absolute value of a measure is a positive measure, we can also express the general definition as follows:

Definition (classical)

The measure μ\mu is absolutely continuous with respect to ν\nu if, for every measurable set AA, |μ|(A)=0{|\mu|}(A) = 0 if |ν|(A)=0{|\nu|}(A) = 0.

Generalisation

Since only the full sets (or, classically, the null sets) matter, we do not need to have the full structure of a measure. Sometimes we equip a measurable space with a δ\delta-filter ℱ\mathcal{F} of full sets (or a σ\sigma-ideal 𝒩\mathcal{N} of null sets) without specifying a measure that produces these. (For example, a smooth manifold is so equipped, effectively the full/null sets under Lebesgue measure, even though there is no canonical such measure, since these sets are the same regardless of coordinate chart.) Then we say:

Definition

The measure μ\mu is absolutely continuous with respect to ℱ\mathcal{F} (or 𝒩\mathcal{N}) if every element of ℱ\mathcal{F} is μ\mu-full (or every element of 𝒩\mathcal{N} is μ\mu-null).

In fact, only the full/null sets of μ\mu matter either, but until somebody has use for the notion of one δ\delta-filter (or σ\sigma-ideal) being absolutely continuous with respect to another, I will refrain from writing it down. (See centipede mathematics.)

Defaults

If one calls a measure on the real line ‘absolutely continuous’, this means with respect to Lebesgue measure.

This generalises to any cartesian space (with Lebesgue measure) or indeed to any smooth manifold of finite dimension (where there is no canonical Lebesgue measure but a family of local ones and so still a notion of Lebesgue-full and Lebesgue-null sets).

Or, this generalises to any compact group (with Haar measure) or indeed to any locally compact group (where there is no canonical Haar measure but a proportional family of them and so still a canonical notion of Haar-full and Haar-null sets).
