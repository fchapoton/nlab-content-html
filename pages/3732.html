This page is a result of the following question originally asked at adjunction:

Eric: Is it true that given categories CC and DD that functors F:C→DF:C\to D and G:D→CG:D\to C form an adjunction if the following diagram commutes for all morphisms f:x→yf:x\to y in CC?

x →f y ↑ ↑ G∘F(y) ←G∘F(f) G∘F(x) \array{ x &\stackrel{f}{\to}& y \\ \uparrow && \uparrow \\ G\circ F(y) &\stackrel{G\circ F(f)}{\leftarrow}& G\circ F(x) }

If so, is that an “if and only if” so that you can define adjunctions this way? If so, this would, in my opinion, be the most digestible way to define adjunctions for “scientists and engineers”.

Definition (Experimental)

Note the Crash! below. This definition is not correct. I think something like this should work, but I need to think about it.

Given categories CC and DD, a pair of functors F:C→DF:C\to D and G:D→CG:D\to C form an adjunction if for every morphism f:x→yf:x\to y in CC, there are component morphisms

η yx:G∘F(x)→y\eta_{y x}: G\circ F(x)\to y

and

η xy:G∘F(y)→x\eta_{x y}: G\circ F(y)\to x

such that the following diagram commutes

x →f y ↑ ↑ G∘F(y) ←G∘F(f) G∘F(x). \array{ x &\stackrel{f}{\to}& y \\ \uparrow && \uparrow \\ G\circ F(y) &\stackrel{G\circ F(f)}{\leftarrow}& G\circ F(x) }.

Note that since G∘F:C→CG\circ F:C\to C is an endofunctor, there are component morphisms

α x:x→G∘F(x)\alpha_x: x\to G\circ F(x)

and

α y:y→G∘F(y)\alpha_y: y\to G\circ F(y)

such that the following diagram also commutes:

x →f y ↓ ↓ G∘F(x) →G∘F(f) G∘F(y). \array{ x &\stackrel{f}{\to}& y \\ \downarrow && \downarrow \\ G\circ F(x) &\stackrel{G\circ F(f)}{\rightarrow}& G\circ F(y) }.

Putting the two commuting squares together, we have a commuting square with commuting (upward) diagonals giving rise to additional commuting triangles.

These commuting diagrams give rise to the following relations:

  • η yx∘α x=f\eta_{y x}\circ \alpha_x = f

  • α y∘η yx=G∘F(f)\alpha_y\circ\eta_{y x} = G\circ F(f)

  • [G∘F(f)]∘α x=α y∘f[G\circ F(f)]\circ\alpha_x = \alpha_y\circ f

  • η xy∘[G∘F(f)]∘α x=1 x\eta_{x y}\circ [G\circ F(f)]\circ\alpha_x = 1_x

  • f∘η xy∘α y=1 y.f\circ\eta_{x y}\circ\alpha_y = 1_y.

Note: I’m not sure if these are independent or not (need to count how many independent equations we get!), but it is instructive to note:

η xy∘α y∘η yx∘α x=1 x\eta_{x y}\circ \alpha_y\circ\eta_{y x}\circ\alpha_x = 1_x

and

η yx∘α x∘η xy∘α y=1 y\eta_{y x}\circ\alpha_x\circ\eta_{x y}\circ \alpha_y = 1_y

which means

η yx∘α x=(η xy∘α y) −1.\eta_{y x}\circ\alpha_x = \left(\eta_{x y}\circ \alpha_y\right)^{-1}.

Response from Toby

You need to say more than just that there are morphisms that make these diagrams commute. Whether F⊢GF \vdash G is an adjunction is not actually a property but a structure; we need to say how they are an adjunction. That is, you have to pick the family of morphism G(F(x))→yG(F(x)) \to y in advance.

But then you can make it work, I believe … although I'm pretty sure that you need more commutative diagrams than the one that you've drawn. That is, an adjunction η\eta from FF to GG consists of a family of maps η f:G(F(x))→y\eta_f\colon G(F(x)) \to y, one for each morphism f:x→yf\colon x \to y, such that

  • a bunch of diagrams commute.

I have to think about whether this works and what diagrams you need.
