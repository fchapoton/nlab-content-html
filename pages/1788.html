In Def. the condition of 0-truncation is crucial in two respects:

  1. First of all it means that 𝒳→𝔹G\mathcal{X} \to \mathbb{B}G is a faithful morphism which identifies it as an object in geometric GG-equivariant homotopy theory via

    (𝒳 ↓ 𝔹G)∈((H sing) /𝔹G) faith≃Sh ∞(GOrbits,H) \left( \array{ \mathcal{X} \\ \downarrow \\ \mathbb{B}G } \right) \;\in\; \left( \left( \mathbf{H}_{sing} \right)_{/\mathbb{B}G} \right)_{faith} \;\simeq\; Sh_\infty\left( G Orbits, \mathbf{H}\right)

  2. but moreover, only with this condition is the shape of an orbifold in the sense of Prop. the expected object in plain GG-equivariant homotopy theory

    ʃ𝒳∈Sh ∞(Singularities,∞Groupoids) /𝔹G⏟≃Sh ∞(GOrbits,∞Groupoids)↪DiscSh ∞(Singularities,H) /𝔹G⏟H sing &#643; \mathcal{X} \;\in\; \underset{ \simeq Sh_\infty\left( G Orbits, \infty Groupoids \right) }{ \underbrace{ Sh_\infty\left( Singularities, \infty Groupoids \right)_{/\mathbb{B}G} } } \overset{Disc}{\hookrightarrow} \underset{ \mathbf{H}_{sing} }{ \underbrace{ Sh_\infty\left( Singularities, \mathbf{H} \right)_{/\mathbb{B}G} } }
