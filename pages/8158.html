Amnestic functors

  • Properties
  • Examples
  • Related concepts
  • References

Definition

An amnestic functor is a functor U:𝒟→𝒞U : \mathcal{D} \to \mathcal{C} (between strict categories) that reflects identity morphisms: more precisely, for any isomorphism f:a→bf\colon a \to b in 𝒟\mathcal{D}, if Ua=UbU a = U b and Uf=id UaU f = id_{U a}, then a=ba = b and f=id af = id_a.

If we follow the principle of equivalence and refuse to state equations between objects, then the closest thing to this that we can say is that if Ua≅UbU a \cong U b (say via g:Ua→Ubg\colon U a \to U b) and Uf=g∘id UaU f = g \circ id_{U a} (which is a trivial hypothesis, since gg can simply be UfU f), then a≅ba \cong b (say via h:a→bh\colon a \to b) and f=h∘id af = h \circ id_a (which is a trivial conclusion, since hh can simply be ff.) Thus, this is really a property of strict functors between strict categories. Or to put it another way, if U:𝒟→𝒞U\colon \mathcal{D} \to \mathcal{C} is any functor, then there exist equivalent categories 𝒟′≃𝒟\mathcal{D}' \simeq \mathcal{D} and 𝒞′≃𝒞\mathcal{C}' \simeq \mathcal{C} such that the composite 𝒟′→𝒟→F𝒞→𝒞′\mathcal{D}' \to \mathcal{D} \overset{F}\to \mathcal{C} \to \mathcal{C}' is amnestic.

Properties

  • An amnestic full and faithful functor is automatically an isocofibration, i.e. injective on objects: if UD′=UDU D' = U D, then there is some isomorphism f:D′→Df : D' \to D in 𝒟\mathcal{D} such that Uf=id UDU f = id_{U D}, but then we must have f=id UD′=id UDf = id_{U D'} = id_{U D}, so D′=DD' = D.

  • An amnestic isofibration has the following lifting property: for any object DD in 𝒟\mathcal{D} and any isomorphism f:C→UDf : C \to U D in 𝒞\mathcal{C}, there is a unique isomorphism f˜:C˜→D\tilde{f} : \tilde{C} \to D such that Uf˜=fU \tilde{f} = f. Indeed, if f˜′:C˜′→D\tilde{f}' : \tilde{C}' \to D were any other isomorphism such that Uf˜′=fU \tilde{f}' = f, then U(f˜ −1∘f˜′)=id CU (\tilde{f}^{-1} \circ \tilde{f}') = id_C, so we must have f˜=f˜′\tilde{f} = \tilde{f}'.

  • If the composite U∘KU \circ K is an amnestic functor, then KK is also amnestic.

Examples

  • Any strictly monadic functor is amnestic. Conversely, any monadic functor that is also an amnestic isofibration is necessarily strictly monadic.
