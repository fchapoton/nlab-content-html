Idea

The hyperbolic functions (or sometimes hyperbolic trigonometric functions) are analogues of the usual trigonometric functions, but adapted to the hyperbola x 2−y 2=1x^2 - y^2 = 1 rather than the circle x 2+y 2=1x^2 + y^2 = 1.

Definition

There are multiple ways of introducing the hyperbolic functions. Probably the most straightforward is to use the following definitions based on the exponential function exp\exp:

  • cosh(x)≔12(exp(x)+exp(−x))\cosh(x) \coloneqq \frac1{2} (\exp(x) + \exp(-x)) (hyperbolic cosine, sometimes pronounced as “kosh”). This can be interpreted as a function ℝ→ℝ\mathbb{R} \to \mathbb{R}, or as a function ℂ→ℂ\mathbb{C} \to \mathbb{C}.

  • sinh(x)≔12(exp(x)−exp(−x))\sinh(x) \coloneqq \frac1{2} (\exp(x) - \exp(-x)) (hyperbolic sine, sometimes pronounced as “cinch”). This also can be interpreted as a function ℝ→ℝ\mathbb{R} \to \mathbb{R}, or as a function ℂ→ℂ\mathbb{C} \to \mathbb{C}.

  • The remaining hyperbolic functions are defined by tanh≔sinhcosh\tanh \coloneqq \frac{\sinh}{\cosh}, coth=coshsinh\coth = \frac{\cosh}{\sinh}, sech=1cosh\sech = \frac1{\cosh}, csch=1sinh\csch = \frac1{\sinh}.

Note that (cosh(t),sinh(t))(\cosh(t), \sinh(t)) (as a pair of real-valued functions) lies on the hyperbola x 2−y 2=1x^2 - y^2 = 1, and in fact this is a parametrization of the hyperbola, much as (cos(t),sin(t))(\cos(t), \sin(t)) parametrizes the unit circle x 2+y 2=1x^2 + y^2 = 1.

It is straightforward to establish the following further properties by exploiting properties of the exponential function:

  • cosh(x+y)=cosh(x)cosh(y)+sinh(x)sinh(y)\cosh(x + y) = \cosh(x)\cosh(y) + \sinh(x)\sinh(y) and sinh(x+y)=sinh(x)cosh(y)−cosh(x)sinh(y)\sinh(x+y) = \sinh(x)\cosh(y) - \cosh(x)\sinh(y) (“addition formulas”),

  • (cosh)′=sinh(\cosh)' = \sinh and (sinh)′=cosh(\sinh)' = \cosh,

  • cosh(x+2πi)=cosh(x)\cosh(x + 2\pi i) = \cosh(x) and sinh(x+2πi)=sinh(x)\sinh(x + 2\pi i) = \sinh(x) for all x∈ℂx \in \mathbb{C},

  • cosh(x)=∑ n≥0x 2n(2n)!\cosh(x) = \sum_{n \geq 0} \frac{x^{2 n}}{(2 n)!} and sinh(x)=∑ n≥0x 2n+1(2n+1)!\sinh(x) = \sum_{n \geq 0} \frac{x^{2 n + 1}}{(2 n + 1)!}.

Alternative presentation

Another avenue is first to introduce the inverse hyperbolic functions as integrals of suitable algebraic functions, e.g.,

sinh −1(t)=∫ 0 tdxy\sinh^{-1}(t) = \int_0^t \; \frac{d x}{y}

where y=x 2+1y = \sqrt{x^2 + 1}. Or, what is essentially the same, construct a solution p(t)p(t) to the differential equation

(p′) 2=p 2+1,p(0)=0(p')^2 = p^2 + 1, \qquad p(0) = 0

so that (p′(t),p(t))(p'(t), p(t)) parametrizes the curve x 2−y 2=1x^2 - y^2 = 1; the approach by invoking an integral is in accordance with an elementary method in differential equations that goes by the name “separation of variables”.

This particular approach is similar to the way that elliptic functions were introduced historically, by studying integrals of algebraic functions

∫dxy\int \; \frac{d x}{y}

where yy is related to xx via an algebraic relation such as y 2=x 3+ax+by^2 = x^3 + a x + b. For suitable such relations these give the various so-called elliptic integrals; for more on what this has to do with ellipses, see Wikipedia, e.g., here. Elliptic functions are then suitable inverses of elliptic integrals, following Jacobi, Abel, and others throughout the nineteenth century (e.g., Weierstrass; see also Weierstrass elliptic function).
