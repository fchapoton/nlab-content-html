This page is about the notion of poly-morphism introduced by Shinichi Mochizuki. For the concept in computer science see polymorphism. Or see universe polymorphism for the concept in type theory.

Idea

For a category CC, a poly-morphism is a collection of morphisms with common source and target, considered as a morphism in a new category C polyC^{poly} with the same objects, but with hom-sets C poly(a,b):=P(C(a,b))C^{poly}(a,b) := P(C(a,b)), the power set of the original hom-sets.

These play a substantive rôle in Shinichi Mochizuki‘s inter-universal Teichmüller theory.

Abstract approach

In the following, fix a monoidal category VV and a VV-enriched category CC. Examples to keep in mind for VV are the category Set of sets considered as a cartesian monoidal category, or the category G-set for a given group GG, again with the cartesian monoidal structure.

Definition

For P:V→VP\colon V\to V a lax monoidal functor, the PP-poly category C PpolyC^{P poly} is the VV-category with the same objects as CC, and where the VV-hom object C Ppoly(a,b):=P(C(a,b))C^{P poly}(a,b) := P(C(a,b)).

That C PpolyC^{P poly} really is a VV-category, i.e. that it has a composition map (that is associative etc) follows from the lax monoidal structure on PP. Composition is defined to be

C Ppoly(a,b)⊗C Ppoly(b,c)=P(C(a,b))⊗P(C(b,c))→P(C(a,b)⊗B(b,c))→P(C(a,c))=C Ppoly(a,c) C^{P poly}(a,b) \otimes C^{P poly}(b,c) = P(C(a,b)) \otimes P(C(b,c)) \to P(C(a,b)\otimes B(b,c)) \to P(C(a,c)) = C^{P poly}(a,c)

and this is associative, using the definitions of lax monoidal functor and enriched category. The unit map

I→P(I)→P(C(a,a))=C Ppoly(a,a) I \to P(I) \to P(C(a,a)) = C^{P poly}(a,a)

satisfies the required conditions again from the properties of PP.

Examples of such functors PP are:

  • For V=SetV=Set with the cartesian monoidal structure, CC an arbitrary category, take P:Set→SetP\colon Set \to Set the covariant power set functor. This is lax monoidal since every pair of subsets U⊆SU \subseteq S and V⊆TV\subseteq T gives a subset U×V⊆S×TU\times V \subseteq S\times T.

  • For GG a group, V=GSetV=GSet with the cartesian model structure, CC a category enriched in GG-sets, let P:GSet→GSetP\colon GSet \to GSet also be the covariant power set functor, but for SS a GG-set, equip P(S)P(S) with the GG-action

    U↦g⋅U={g⋅x∈S∣x∈U}. U\mapsto g\cdot U = \{g\cdot x \in S \mid x\in U\}.

When we are in the first case, V=SetV=Set and PP the power set functor, so that CC is a plain category, then we shall refer to C PpolyC^{P poly} as just C polyC^{poly}, and its morphisms as poly-morphisms.

A poly-isomorphism of CC is defined to be a poly-morphism of the core of CC, so that it is a collection of invertible arrows of CC. Note that these are not the isomorphisms in C polyC^{poly}: all the isomorphisms in C polyC^{poly} come from CC.
