Idea

Roughly speaking, a C *C^*-correspondence between two C *C^*-algebras AA and BB is a generalised C *C^*-homomorphism from AA to BB.

Definition

Definition

Let AA and BB be C *C^*-algebras. An A,BA,B-correspondence is a pair (E,φ)(E,\varphi) consisting of a right Hilbert BB-module EE and a non-degerate C *C^*-homomorphism φ:A→ℒ B(E)\varphi \colon A\to \mathcal{L}_B(E); i.e., the image φ(A)E\varphi(A)E is dense in EE (as Hilbert BB-modules).

We write

A→(E,φ)B, A\stackrel{(E,\varphi)}{\to} B,

or just A→EBA\stackrel{E}{\to}B, and call this a C *C^*-correspondence.
