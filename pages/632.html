Definition

Definition

A functor F:C→DF:C\to D is conservative if it is “isomorphism-reflecting”, i.e. if g:a→bg:a\to b is a morphism in CC such that F(g)F(g) is an isomorphism in DD, then gg is an isomorphism in CC.

Remark

Sometimes conservative functors are assumed to be faithful as well. If CC has, and FF preserves, equalizers, then conservativity implies faithfulness.

See conservative morphism for a generalization to an arbitrary 2-category.

Examples

Proposition

Let 𝒞\mathcal{C} be a category with pullbacks. Given any morphism f:X⟶Yf \colon X \longrightarrow Y in 𝒞\mathcal{C} write

f *:𝒞 /Y⟶𝒞 /X f^\ast \colon \mathcal{C}_/Y \longrightarrow \mathcal{C}_{/X}

for the functor of pullback along ff between slice categories (base change). If strong epimorphisms in 𝒞\mathcal{C} are preserved by pullback, then the following are equivalent:

  1. ff is a strong epimorphism;

  2. f *f^\ast is conservative.

(e.g. Johnstone, lemma 1.3.2)

Example

Every fully faithful functor is a conservative functor.

Properties

Proposition

A conservative functor F:C→DF : C \to D reflects all limits and colimits that it preserves and which exist in the source category.

Proof

Let K:J→CK : J \to C be a diagram in CC whose limit limK\lim K exists and such that limF∘K≃FlimK\lim F\circ K \simeq F \lim K. Then if const c→Kconst_c \to K is a cone in CC that is sent to a limiting cone Fconst cF const_c in DD, then by the universal property of the limit in DD the morphism F(c→limK)F( c \to \lim K) is an isomorphism in DD, hence must have been an isomorphism in CC, hence const cconst_c must have been a limiting cone in CC.

The arguments for colimits is analogous.
